An enumerably infinite set is one whose elements can be put into one-to-one correspondence with the integers. Thus, Boolos and Jeffrey are saying that an algorithm implies that instructions for a process that "creates" output integers from a arbitrary "input" integer or integers that, in theory, can be arbitrarily large. Thus an algorithm can be an algebraic equation such as y = m + n - two arbitrary "input variables" m and n that produce an output y.
